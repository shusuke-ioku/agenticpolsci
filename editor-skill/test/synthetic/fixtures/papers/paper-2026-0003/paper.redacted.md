# Replicating Putnam (1993) on Regional Institutional Performance

**Author:** [redacted]

## Abstract

We attempt to replicate Putnam's (1993) analysis of Italian regional
government performance using reconstructed data from the original surveys.
Our reproduction succeeds for 8 of 12 published tables; the remaining 4
tables' results differ from the published values by margins exceeding
rounding tolerance. We identify the discrepancy as a coding error in our
own pipeline (see §4).

## 1. Setup

We obtain survey-level data from the ICPSR archive. We reconstruct Putnam's
composite "institutional performance index" from the 12 underlying indicators.

## 2. Tables 1-8

These reproduce within expected tolerance.

## 3. Tables 9-12

Our computed values diverge from Putnam's published figures. On investigation
we identify a coding error in our pre-processing pipeline (a sign error on
one of the three budget-process indicators). We document this but cannot
resolve it without access to the original author's cleaning code.

## 4. Code error

We attempted to contact the author. No response within the submission
window. We therefore submit this replication flagging the failure.

## 5. Conclusion

Partial replication. Tables 9-12 do not reproduce; the responsible error
is in our own pipeline but we cannot fix it without the original author's
code. We submit to document the failure for future replicators.
